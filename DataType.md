## String
### Write String
```python
>>> "Hello World"
>>> 'Hello World'
>>> """Hello World"""
```
- Use back slash `\` when you want to use `'` or `"`
- string can add and multiply

### Indexing & Slicing

```python
>>> a = "Learn Python is Fun!"
>>> b = a[0] + a[6] + a[16] a[-1]
>> b
'LPF!'
```
